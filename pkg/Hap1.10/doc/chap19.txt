  
  [1X19 Cat-1-groups[0X
  
  ::::::::::::::::::::::::
  [10XAutomorphismGroupAsCatOneGroup(G)[0m
  
  Inputs  a group G and returns the Cat-1-group C corresponding to the crossed
  module G-> Aut(G).
  ::::::::::::::::::::::::
  [10XHomotopyGroup(C,n)[0m
  
  Inputs  a  cat-1-group C and an integer n. It returns the nth homotopy group
  of C.
  ::::::::::::::::::::::::
  [10XHomotopyModule(C,2)[0m
  
  Inputs  a  cat-1-group  C and an integer n=2. It returns the second homotopy
  group  of  C  as  a  G-module  (i.e.  abelian  G-outer group) where G is the
  fundamental group of C.
  ::::::::::::::::::::::::
  [10XQuasiIsomorph(C)[0m
  
  Inputs  a  cat-1-group  C and returns a cat-1-group D for which there exists
  some homomorphism C-> D that induces isomorphisms on homotopy groups.
  
  This function was implemented by [12XLe Van Luyen[0m.
  ::::::::::::::::::::::::
  [10XModuleAsCatOneGroup(G,alpha,M)[0m
  
  Inputs  a  group G, an abelian group M and a homomorphism alpha: G-> Aut(M).
  It returns the Cat-1-group C corresponding th the zero crossed module 0: M->
  G.
  ::::::::::::::::::::::::
  [10XMooreComplex(C)[0m
  
  Inputs a cat-1-group C and returns its Moore complex as a G-complex (i.e. as
  a complex of groups considered as 1-outer groups).
  ::::::::::::::::::::::::
  [10XNormalSubgroupAsCatOneGroup(G,N)[0m
  
  Inputs  a  group  G  with  normal  subgroup  N. It returns the Cat-1-group C
  corresponding th the inclusion crossed module N-> G.
  ::::::::::::::::::::::::
  [10XXmodToHAP(C)[0m
  
  Inputs  a  cat-1-group  C  obtained  from  the  Xmod  package  and returns a
  cat-1-group D for which IsHapCatOneGroup(D) returns true.
  
  It returns "fail" id C has not been produced by the Xmod package.
